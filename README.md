# Functional Requirements

## User Management
- The system shall allow users to register, log in, and log out.
- The system shall allow authenticated users to view their own profile.
- The system shall use cookies for authentication, with cookies expiring after five (5) minutes.

## Public Access
- The system shall allow unauthenticated users to view available apartments.
- The Available Apartments page shall display only listings whose available-from date is later than the current date.

## Apartment & Listing Viewing
- The system shall allow users to view available apartments.
- The system shall allow administrators to view all apartments in the system.
- The system shall allow administrators to view apartment details.
- The system shall allow administrators and property managers to view all listings in the system.

## Administrator Capabilities
- The system shall allow administrators to view all property managers in the system.
- The system shall allow administrators to view property manager profiles.
- The system shall allow administrators to view all tenants in the system.
- The system shall allow administrators to view tenant profiles.
- The system shall allow administrators to view all buildings and their associated properties.
- The system shall allow administrators to modify building properties.
- The system shall allow administrators to modify listings.

## Property Manager Capabilities
- The system shall allow property managers to view buildings and their associated properties.
- The system shall allow property managers to view apartments and their associated properties.
- Building & Apartment Management
- The system shall allow administrators and property managers to create buildings.
- The system shall allow administrators and property managers to create apartments.

## Listing Management
- The system shall allow administrators and property managers to create, modify, and delete listings.

## Owner Capabilities
- The system shall allow owners to view all property managers and tenants in the system.
- The system shall allow owners to create, edit, and delete property manager accounts.
- The system shall allow owners to create, edit, and delete tenant accounts.
